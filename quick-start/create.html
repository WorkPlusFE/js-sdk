# 创建应用

此文档以WorkPlus v4.0新版本为主，会与3.0版本的交互有所不同，但创建应用所需要设置的内容基本一致。同时，此文档只描述轻应用相关的内容。

# 添加内部应用

登录管理后台，选择并进入对应的组织，然后点击应用管理菜单，从右上角的添加按钮，选择内部应用。
