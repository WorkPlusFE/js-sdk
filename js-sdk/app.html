# 路由跳转

根据参数跳转到指定的 App。

使用说明

客户端 Android iOS
支持情况 支持 支持
使用 WorkPlus 扫一扫
w6s.app.route();

参数说明

参数 类型 说明
scheme_url String 除了原生应用的schema_url规则, 也支持内部的应用协议, 例如workplus://openApp?id=abc&domainId=abc
android_explicit_intent.package String 包名
android_explicit_intent.init_url String 跳转的具体页面
android_explicit_intent.params Object 携带的参数

android_explicit_intent 仅 Android 客户端支持。
