# 当前连接Wifi信息

获取当前连接 Wifi 信息, 包括 bssid、名字等。

使用说明

客户端 Android iOS
支持情况 支持 支持
使用 WorkPlus 扫一扫
w6s.network.getWifiInfo({
  success: function(res) {},
  fail: function(err) {},
});

返回数据

参数 说明
bssid wifi 接入点标识
name wifi 名称

提示

同一个路由设备，bssid可能会有多个不同的值。
