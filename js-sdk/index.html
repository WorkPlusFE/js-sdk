# 概述

通过使用 WorkPlus JS-SDK，轻应用开发者可高效地使用拍照、选择照片、位置等手机系统的能力，同时可以直接使用分享、扫一扫及选择人员等定制能力，为用户提供更优质的应用使用体验。

特性：

  • 开箱即用，cordova.js自动注入，支持数据模拟；
  • 所有方法无须等待deviceready，支持Promise及callback的调用方式；
  • 支持script标签直接引入，支持基于webpack等工具的按需引入；
  • 基于TypeScript编写，质量优良、提示完善；
  • 内含10+个功能模块，轻松应付各种业务需求。
