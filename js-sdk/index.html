# 概述

WorkPlus JS-SDK 为H5轻应用提供了调用原生控件的能力，帮助开发者高效使用拍照、定位等手机系统的能力，同时可以直接使用扫一扫、分享等定制能力，带给轻应用接近原生的体验。

特性：

  • 开箱即用，cordova.js智能注入，支持模拟数据；
  • 所有方法无须等待deviceready，支持Promise及callback的调用方式;
  • 支持直接script标签引入，支持基于webpack等工具的按需引入；
  • 基于TypeScript编写，质量优良、提示完善；
  • 10+ 个模块，满足绝大部分需求场景。
