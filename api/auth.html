# 退出登录

通过轻应用告知当前 accessToken 过期，WorkPlus 将退出当前登录状态。

警告

从 H5 的角度，强制退出登录，是属于比较容易让用户产生误解的行为，请慎用该方法！！

使用说明

客户端 Android iOS
支持情况 支持 支持
使用 WorkPlus 扫一扫
w6s.auth.onAccessTokenOverdue();
