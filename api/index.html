# 概述

WorkPlus是企业移动通信平台和企业移动应用平台的有机统一。它是“微信企业版”，与微信不同，它为企业内部员工之间的安全通信提供了强力的支持，同时也是企业移动应用的统一入口。它的一些典型特征如下：

  • 单聊、群聊
  • 文本消息、图片消息、语音消息、视频消息、图文消息
  • 即时消息、离线消息
  • 服务号（相当于微信的公众号）
  • 同事圈（相当于微信的朋友圈）
  • 企业组织架构与企业通信录
  • 常用联系人、好友关系
  • 企业移动应用中心，支持轻应用和原生应用的接入，单点登录
  • 应用收发消息和访问通信录支持
  • 移动APP、PC客户端
  • 管理平台
  • 开放API
